버스 도착 정보

경기도 버스 공공 API를 사용해요.
아래 안내를 따라 인증키를 발급받아 입력해 주세요.

data.go.kr → 로그인 → "경기도 버스도착정보 조회 서비스" 검색 → 활용신청 → 마이페이지에서 일반 인증키(디코딩) 복사

API 키 발급 방법
1
data.go.kr에 회원가입 후 로그인
2
검색창에 "경기도 버스도착정보 조회" 검색
3
오픈 API 선택 → 활용신청 (즉시 승인)
4
마이페이지 → API 목록 → 일반 인증키(디코딩) 복사 후 위에 붙여넣기
